TacoTek
=======

A Minecraft mod which aims to inspire you to world Domination!


Features:
--------------
- [x] Fully craft-able Tacos!
- [x] Taco Crates!
- [ ] Flying Toasters!
- [ ] Jetpacks with built in batpacks!
- [ ] Armor with built in batpacks!
- [ ] Exterior & realistic reactor cooling!
