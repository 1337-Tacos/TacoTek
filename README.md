TacoTek
=======

A Minecraft mod which aims to inspire you to world Domination!


Features:
--------------
- [x] Fully craft-able Tacos!
- [x] Taco Crates!
- [ ] Non-OP Modular Armor!
- [ ] Modular Armor that requires a brain!
- [ ] Armor Shield Modules!
- [ ] Armor with built in batpacks!
- [ ] Flying Toasters!
